Config Parameters

1. Audio File Path

All audio files are stored in the following path on Azure Blob Storage:

.../nttt/r1_audio/

2. Configuration File Path

Configuration files are located at:

.../nttt/r1_configuration/

3. Fair Use Summary

  • 1 : All audio is streamed and not downloadable.
  • 2 : Is used for stylistic analysis, training, and education.
  • 3 : Compares tempo, styles, chord progressions, and musical themes
  • 4 : Only uses short random snippets of audio.

4. Logic Summary

  • Audio Files: Accessed from the r1_audio container.
  • Configuration Files: Retrieved from the r1_configuration container.
  • SongID: Deterministic UUID v5, generated from the cleaned album name and song title.

5. Metadata Fields

  • TIT2 - Title: The song title.
  • TALB - Album: The album name.
  • TPE1 - Artist: The matched artist.
  • TCON - Style: Genre (e.g., Tango, Vals, Milonga).
  • COMM - Comment: Includes the generated SongID.
